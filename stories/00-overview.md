## Priority Legend

- **P0**: Required for MVP / App Store launch
- **P1**: Important, do after launch
- **P2**: Nice to have, future roadmap

## MVP Scope (P0 Stories)

### Must Have for App Store Launch:
1. Epic 1: Multi-Car Support (all stories)
2. Epic 2: Mode Configuration + Simple Mode + Business Mode
3. Epic 3: Subscriptions (RevenueCat, paywall, free limits)
4. Epic 4: Onboarding (welcome, account, mode, car, permissions)
5. Epic 5: Core Trip Tracking (existing, verify with multi-car)
6. Epic 6: Dashboard Stats (basic)
7. Epic 7: CSV Export
8. Epic 9: Essential Settings

### Post-Launch (P1):
- Diary Mode features
- Advanced insights
- Premium reports
- Web dashboard enhancements

### Future (P2):
- Calendar integration
- Photos integration
- Family plan features
- B2B features

## Estimation Guide

| Size | Description | Rough Effort |
|------|-------------|--------------|
| XS | Trivial change | < 2 hours |
| S | Small feature | 2-4 hours |
| M | Medium feature | 1-2 days |
| L | Large feature | 3-5 days |
| XL | Epic/complex | 1-2 weeks |

- Implementation details

- Design guidance
